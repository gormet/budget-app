STEP 1: Create GitHub Repository
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://github.com/new

2. Fill in:
   Repository name: budget-app
   Description: Multi-user budgeting app with workspaces
   Visibility: [Private or Public - your choice]

3. ⚠️ IMPORTANT: Do NOT check these boxes:
   ❌ Add a README file
   ❌ Add .gitignore
   (We already have them!)

4. Click "Create repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2: Push Your Code
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Copy these commands one by one:

# Replace YOUR_USERNAME with your GitHub username!

git remote add origin https://github.com/YOUR_USERNAME/budget-app.git
git branch -M main
git push -u origin main

EXAMPLE (if your username is "septian"):
git remote add origin https://github.com/septian/budget-app.git
